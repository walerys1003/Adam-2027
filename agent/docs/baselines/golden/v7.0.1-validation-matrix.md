### Latency notes (read before treating these as hard SLAs)

- `avg_turn_latency_ms` is full-turn (caller speech end → agent audio), over a **real PSTN trunk to
  a cloud provider**, not a LAN benchmark. Treat these as field-observed envelopes, not lab targets.
- Cleanest fast lane: **grok3 (692 avg / 708 max)** and **google_live (515 / 548)** — both well inside the
  sub-1s full-agent budget.
- **deepgram** max 3589 ms is a single outlier turn (avg 1543); **openai_realtime** and **elevenlabs**
  sit in the 1.6–2.1s avg band — above the nominal sub-1s full-agent budget in CLAUDE.md, but consistent
  for cloud round-trips at this hour. Flagged for awareness, not a blocker.
- **local_hybrid 3984** is a *single-turn* call (`total_turns=1`); that one turn absorbed the cold-start
  STT reconnect (~10s span), so avg=max here is not representative of warm steady-state. Do **not** anchor
  the hybrid latency floor on this number — re-measure with a multi-turn warm call.

## Known transient (documented so it is NOT re-flagged as a new bug)

**Local Hybrid first-turn STT WebSocket handshake timeout.** On call 5, the local STT adapter
(`ws://127.0.0.1:8765`) handshake hit the 3s `asyncio.wait_for` ceiling at t+3s
(`STT open_call failed` → `TimeoutError`), the reconnection/resilience layer retried, and the
session opened on the next attempt (~t+12s). Call completed, `error_message=NULL`.

- **Not a v7.0.1 regression:** `git log feat/7.0.1 ^origin/main -- src/pipelines/local.py` is empty
  (adapter untouched this release). The retry path predates 7.0.1 (commit `343ceed7`, "reconnection
  logic and error resilience to local adapter").
- **Historical parity:** all 18 prior `demo_hybrid` calls in `call_records` (back to 2026-04) ended
  `completed`/`transferred` — same outcome. The timeout is only *visible* now because this run captured
  per-call streaming logs that prior runs did not.
- **If chasing it later:** likely a cold-start race / momentary port contention on the local AI server
  WS. Candidate follow-up = widen the STT open timeout or warm the local adapter on engine start. Tracked
  as a non-blocking nice-to-have, not a 7.0.1 gate.
